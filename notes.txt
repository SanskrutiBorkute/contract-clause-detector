1. Project Overview
   =========================================================

ClauseGuard AI is an AI-powered contract analysis system developed using
Python, Flask, Natural Language Processing (NLP), and Machine Learning.

The application allows users to upload legal contracts in PDF, DOCX,
or image format. The system automatically extracts text, segments the
contract into clauses, identifies risky and predatory clauses, detects
missing legal protections, calculates an overall risk score, and
generates a downloadable PDF analysis report.

=========================================================
4. Project Installation
=======================

Step 1

Clone the repository.

Step 2

Create Virtual Environment

python -m venv .venv

Step 3

Activate Virtual Environment

Windows

.venv\Scripts\activate

Step 4

Install project dependencies

pip install -r requirements.txt

Step 5

Run the project

python app.py

The Flask server will start successfully.
